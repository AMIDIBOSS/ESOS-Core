Welcome to ESOS

Embedded Service Operating System

ESOS is a lightweight operating system desinged to preform simple tasks like file managment.

Overview

ESOS is Very lightweight! It only requires:
  • Single core 500Mhz CPU
  • 128Mb of ram
  • ANY graphics (for setup)
  • ANY bootable media (USB stick, Disk drive)
  • Keyboard
AND that's not minimum! That's ALL IT NEEDS!*Additionally we would recomend any kind of writable filesystem. And a mouse.

Getting started

You can find it's Source Code on github. Fork it or clone it. We don't provide downloads Yet! You have to build it youself!
